# B站学习专注提醒助手 (BiliBili Study Focus Reminder Assistant)

一个Tampermonkey脚本，在Bilibili视频页面上提供渐进式、非侵入式的专注干预，帮助用户在设定的学习时段内保持专注。

## 功能特性

### 🎯 核心功能
- **学习时段管理**：自定义学习时间段，只在指定时段内启用专注提醒
- **白名单系统**：将学习相关视频加入白名单，避免不必要的干预
- **渐进式干预**：根据分心时长逐步升级干预强度，从视觉提示到单词验证
- **实时状态监控**：浮动窗口显示当前学习/分心状态和时长统计
- **暗色模式支持**：完整的暗色主题，支持明暗主题切换

### 📊 数据统计
- **学习时间追踪**：精确记录有效学习时间和分心时间
- **单词学习系统**：内置中英单词验证，帮助学习词汇
- **历史趋势分析**：7天学习趋势图表，可视化学习进度
- **详细统计面板**：包含5个模块的完整学习数据展示

### 🎨 用户体验
- **可拖拽浮动窗口**：自由调整位置，不遮挡视频内容
- **全屏自动隐藏**：进入全屏模式时自动隐藏浮动窗口
- **非侵入式设计**：不影响Bilibili核心功能，仅在学习时段内干预
- **数据持久化**：所有设置和统计数据通过localStorage保存

## 安装使用

### 前置要求
1. 安装浏览器扩展：[Tampermonkey](https://www.tampermonkey.net/)
2. 支持的浏览器：Chrome 90+、Edge 90+（Chromium内核）

### 安装步骤
1. 下载 `bilibili-study-focus-assistant.user.js` 文件
2. 打开Tampermonkey管理面板
3. 点击"添加新脚本"或直接将文件拖入
4. 保存并启用脚本

### 配置说明

脚本提供两种配置方式：

**方式一：参数设置面板（推荐）**

在详细统计面板 header 中点击 ⚙️ 设置 按钮，即可可视化编辑所有参数，无需修改脚本源码。支持学习时段、白名单、词库的增删改查，带实时验证和保存反馈。

**方式二：手动编辑脚本配置**

脚本顶部有详细的用户配置区域，可以自定义以下参数：

```javascript
// 学习时段：格式为 [["开始时间", "结束时间"], ...]
studyPeriods: [
    ["08:00", "12:00"],
    ["14:00", "18:00"],
    ["19:00", "22:00"]
],

// 白名单：学习相关视频的BV号
whitelist: {
    "BV1EpNjzgEcs": { name: "我的学习视频", addedAt: Date.now() }
},

// 干预阶段配置（单位：秒）
interventionStages: [
    { threshold: 0, interval: 0 },      // 阶段0：确认弹窗
    { threshold: 60, interval: 0 },     // 阶段1：1分钟，仅视觉提示
    { threshold: 300, interval: 120 },  // 阶段2：5分钟，弹窗每2分钟
    { threshold: 600, interval: 30 },   // 阶段3：10分钟，弹窗+单词验证每30秒
    { threshold: 1200, interval: 15 }   // 阶段4：20分钟，弹窗+严格验证每15秒
],

// 单词库：格式为 ["中文:英文", ...]
// v1.0.8 扩充至362个六级/考研核心词汇
vocabulary: [
    "学习:study",
    "专注:focus",
    // ... 共362个词条，详见脚本源码
],

// 单词掌握阈值
// 连续答对5次 → 熟悉（权重0.2），连续答对8次 → 已掌握（不再出现）
masteryThreshold: 8,
familiarThreshold: 5,

// 浮动窗口设置
floatingWindow: {
    enabled: true,
    defaultPosition: { x: 20, y: 100 },
    showStats: true
}
```

## 使用指南

### 基本使用
1. 访问Bilibili视频页面（URL匹配 `*://www.bilibili.com/video/BV*`）
2. 浮动窗口会自动出现在页面右上角
3. 绿色表示"学习中"，红色表示"分心中"
4. 点击浮动窗口打开详细统计面板

### 详细统计面板
面板包含5个模块：
1. **今日概览**：学习时间、分心时间、分心次数、单词正确率
2. **当前状态**：当前视频、白名单状态、干预阶段、操作按钮
3. **单词学习**：单词掌握进度、已掌握单词列表、最近答题记录
4. **专注建议**：基于学习数据的个性化建议
5. **历史趋势**：7天学习时间趋势图表

### 干预系统
当观看非白名单视频时，系统会根据分心时长逐步升级干预：

| 阶段 | 分心时长 | 干预措施 | 弹窗间隔 | 新增功能 |
|------|----------|----------|----------|----------|
| 0 | 0秒 | 确认弹窗 | 仅一次 | 课程选择菜单 |
| 1 | 1-5分钟 | 视觉提示 | 无弹窗 | - |
| 2 | 5-10分钟 | 视觉+弹窗+单词填空 | 2分钟 | 单词填空练习 |
| 3 | 10-20分钟 | 视觉+弹窗+单词验证 | 30秒 | 单词验证 |
| 4 | 20分钟+ | 视觉+弹窗+严格验证 | 15秒 | 严格验证 |

### 单词学习系统
- 从配置的单词库中随机选择单词进行验证
- **渐进掌握机制**：连续答对5次降频（熟悉），连续答对8次移入已掌握
- 答错重置连续正确计数；提示后答对不计入连续正确
- 加权随机选词：熟悉词权重0.2，普通词权重1.0
- **刷新词库 / 重置记录**：分别查看词库变更和清除所有学习记录
- 所有答题记录都会被保存和统计
- **新增单词填空练习**：在Stage 2干预中，将"返回学习"按钮替换为单词填空练习，渐进式揭示字母提示

## 技术架构

### 模块设计
```
┌─────────────────────────────────────────────┐
│              Tampermonkey脚本               │
├─────────────────────────────────────────────┤
│  ConfigManager  │  PageMonitor  │  Storage  │
├─────────────────────────────────────────────┤
│  FloatingWindow │  DetailPanel  │  WordVerifier │
├─────────────────────────────────────────────┤
│  InterventionController │  StatisticsTracker │
└─────────────────────────────────────────────┘
```

### 数据存储
- **userConfig**：用户配置，包括学习时段、白名单、干预设置等
- **timeStats**：时间统计数据，包括学习时间、分心时间、分心次数等
- **wordRecords**：单词学习记录，包括掌握状态、答题历史等

### 兼容性
- **完全支持**：Chrome 90+、Edge 90+（Chromium内核）
- **不支持**：Firefox（Tampermonkey API差异）、移动端浏览器
- **未测试**：Safari、其他浏览器

## 常见问题

### Q: 脚本不生效怎么办？
A: 检查以下事项：
1. Tampermonkey扩展是否已启用
2. 脚本是否已启用
3. 当前URL是否匹配 `*://www.bilibili.com/video/BV*`
4. 当前时间是否在学习时段内

### Q: 如何添加视频到白名单？
A: 有两种方式：
1. 在脚本配置中手动添加BV号
2. 在详细统计面板中点击"添加到白名单"按钮

### Q: 数据会丢失吗？
A: 数据通过localStorage保存，在以下情况可能会丢失：
1. 清除浏览器缓存
2. 使用隐私/无痕模式
3. 浏览器存储空间不足

### Q: 可以自定义干预效果吗？
A: 可以，在脚本配置中修改 `interventionStages` 数组，调整阈值和弹窗间隔。
